# 지식 베이스 인덱스

에이전트와 스킬이 조사 결과를 재사용하기 위한 공유 저장소입니다.
새 항목을 추가할 때는 해당 파일을 직접 편집하고 이 인덱스도 업데이트하세요.

## 파일 목록

| 파일 | 내용 | 항목 수 | 마지막 업데이트 |
|------|------|---------|----------------|
| `rss-feeds.md` | 검증된 무료 RSS 피드 | 5 | 2026-02-21 |
| `free-apis.md` | 검증된 무료 API | 2 | 2026-02-21 |
| `job-patterns.md` | job 구현 패턴 및 교훈 | 1 | 2026-02-21 |

## 태그 인덱스

빠른 검색을 위한 태그별 분류입니다.

- `#경제` → rss-feeds.md (CNBC, MarketWatch, Google News)
- `#뉴스` → rss-feeds.md
- `#환율` → free-apis.md (ExchangeRate-API)
- `#AI요약` → free-apis.md (Gemini)
- `#이메일` → job-patterns.md
- `#Gmail` → job-patterns.md

## 사용 방법

- **조회**: `/lookup [키워드]` 또는 에이전트가 자동으로 참조
- **저장**: `/save-research [주제]` 또는 researcher 에이전트가 자동 저장
- **직접 편집**: 파일을 열어 표에 행 추가 후 이 인덱스의 항목 수·날짜 갱신
